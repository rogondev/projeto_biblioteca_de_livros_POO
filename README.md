# 📚 Projeto Biblioteca de Livros - POO

Projeto desenvolvido em Java com foco em Programação Orientada a Objetos, utilizando como base as aulas do Projeto One Alura.

## 🚀 Funcionalidades

- Cadastro de livros físicos
- Cadastro de ebooks
- Sistema de leitura
- Sistema de curtidas
- Classificação automática baseada em leituras

## 🧠 Conceitos aplicados

- Encapsulamento
- Herança
- Polimorfismo
- Sobrescrita de métodos
- Organização em pacotes

## 🛠 Tecnologias utilizadas

- Java 17+
- IntelliJ IDEA
